# AI201 Project 3

## Project Overview

This project builds a text classification model for posts from r/LetsTalkMusic, assigning each post into one of four categories:

- Review: evaluates or critiques an artist, album, or song
- Discussion: shares opinions or expereinces to spark conversation
- Question: asks for information, explanations, or community responses
- Analysis: develops reasoning, arguments, or broader music-related interpretations
